# Available GPU Devices

Use `--device` to select the GPU for evaluation or profiling.

## Device Names

- `nvidia/L40S` (default)
- `nvidia/H100`

## Choosing a Device

- Use `nvidia/L40S` for routine checks.
- Use `nvidia/H100` when you need results for an H100 target environment.
- Keep device choice consistent across comparison runs.

See **hardware/NVIDIA.md** for full device-selection notes.
